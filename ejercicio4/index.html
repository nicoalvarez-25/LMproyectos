Mi infancia

Nací y me crié en Lugo, un hijo único que siempre tuvo el amor incondicional de mis padres. Mis mejores recuerdos son de las tardes jugando en el parque de Frigsa con mis amigos, un grupo inseparable que hacía cualquier día especial. Rodeado de una familia que me apoyó en todo momento, mi infancia fue tranquila y feliz, llena de calor de hogar y paseos por la ciudad. Recuperar contraseña de usuario de Windows

Nota: Esta es una práctica que realicé yo.
El video referenciado a continuación es un recurso externo que ilustra el procedimiento.Esta metodología se basa en sustituir el archivo de Teclas Adhesivas (sethc.exe) por la Terminal de Comandos (cmd.exe) para lograr ejecutar comandos antes del inicio de sesión.El procedimiento inicia accediendo a las Opciones Avanzadas de la ventana de Recuperación de Windows para abrir la terminal. Desde ahí, se usa el Bloc de Notas para navegar a la ruta C:\Windows\System32. En esta ubicación, se localiza el archivo sethc.exe y se le cambia el nombre a sethc.bak (copia de seguridad). Inmediatamente, se localiza cmd.exe, se copia y la copia se renombra como sethc.exe.Al salir y reiniciar el sistema, el usuario debe pulsar la tecla $\text{Shift}$ cinco veces en la pantalla de inicio de sesión. Esto abrirá la terminal (gracias a la sustitución). Finalmente, se utiliza el comando net user [NombreDeUsuario] [NuevaContraseña] para cambiar la contraseña del usuario sin necesidad de ingresar al sistema.

Instalar ubuntu server y windows en la misma máquina virtual

  1. Configuración de la MV

    Crea una sola MV con las siguientes especificaciones: RAM: 4 GB, CPU: 2 procesadores y un Disco Duro de 40 GB (VDI).

  2. Proceso de Instalación

    Se recomienda instalar Windows primero porque su gestor de arranque es menos flexible que el de Linux:

    1. Instalar Windows: Arranca la MV con la ISO de Windows. Durante el particionamiento, debes reducir la partición de Windows para dejar al menos 15-20 GB de espacio SIN ASIGNAR para Ubuntu Server. Finaliza la instalación de Windows.
    2. Instalar Ubuntu Server: Carga la ISO de Ubuntu. En el paso de particionamiento, elige la opción "Manual".
    3. Particionar Linux: Utiliza el espacio sin asignar para crear las particiones de Linux: una partición Raíz (/) (formato Ext4) y una partición swap (intercambio).
    4. Gestor de Arranque: El instalador de Ubuntu detectará Windows y configurará automáticamente el Gestor de Arranque GRUB.

    Al finalizar, cuando inicies la MV, el menú GRUB aparecerá para que elijas arrancar Windows o Ubuntu Server.
